Open Telco

A suite of telco-specific benchmarks for evaluating AI models on telecommunications tasks.

Built on Inspect AI, Open Telco provides standardized evaluations for knowledge, reasoning, and operational capabilities in the telecom domain.

Quick Start‚Äã

Get running in under 5 minutes:

1

Clone the repository

Get the Open Telco codebase from GitHub.

git clone https://github.com/gsma-research/open_telco.git
cd open_telco
2

Install dependencies

Use uv for fast, reliable package management.

pip install uv
uv sync
3

Configure your environment

Add your API keys to a .env file.

# Create .env with:
HF_TOKEN=your_huggingface_token
OPENAI_API_KEY=your_openai_key
4

Run your first evaluation

Test with TeleQnA benchmark.

uv run inspect eval src/open_telco/teleqna/teleqna.py --model openai/gpt-4o --limit 20

For detailed setup instructions, see the Installation Guide.
